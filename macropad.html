Macropad

Summary: During my time as a new hire for Texas Inventionworks, I was given a new hire project of creating a macropad. I was extremely excited to start this project since I had already been planning on creating a macropad for personal use and had sketches already made. This expedited the design process and motivated me to make this macropad satisfactory enough for personal use.

Problem Statement: Create a macropad with the capabilities of typing to pass two test and a competition: first test is to type a predetermined string, the second test is to type a surprise string, and the competition is to create the most "attractive" macropad.

Project Timeline: One month with be given to complete this project with a progress check during the middle of those two months. My personal goal was to have the CAD and electronics done, so I could spend the next half of the time working on the aesthetics.

Design Process: For this project, I wanted to create something that looked a bit different than what was on the market. I wanted to make the footprint as small as possible to minimize desk space, and create an almost floating effect with the keys. We were also given a Raspberry Pi Pico, so I made sure to include space in the CAD for an adapter to switch the Micro-USB to USB-C which I purchased for around a dollar on AliExpress, my source for a lot of the electronics I use. For the material, I opted to use PETG for the plate and a very tight press fit. For the keys, I used resin from the SLA printers at Texas Inventionworks that would retain high dimensional accuracy for the connections between the keys and the switches. The main body was printed from PLA to accommodate the heated inserts I used to secure the plate.

Future Considerations: This project was extremely interesting and I hope to further this design in the future. by adding more features. I also hope to create a smaller main body footprint that would make the design even sleeker. In addition, for the competition, I had overlooked how people enjoyed uniqueness. This led me to lose the competition despite creating something that may have been more practical.

Skills: CAD, Design for Assembly (DFA), Electronics Assembly, and Product Design
